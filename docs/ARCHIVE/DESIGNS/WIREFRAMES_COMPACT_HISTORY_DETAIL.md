# 🔷 1. หลักการออกแบบ Compact

| # | หลักการ | คำอธิบาย |
|---|---------|----------|
| 1 | **Primary Action First** | Section ที่ต้อง action อยู่ด้านบน หลัง header — ไม่ต้อง scroll |
| 2 | **Collapse by Default** | Section ที่เป็น reference / read-only → collapsed, แสดง 1-line summary |
| 3 | **Role-Aware Content** | แต่ละ role เห็นเฉพาะ section ที่เกี่ยวข้องกับ workflow step ของตน |
| 4 | **Unified Section Card** | ทุก section ใช้ format เดียวกัน: Header + Badge + Summary + Expandable Body |
| 5 | **Balance Inline** | Balance Cards จาก 8 cards (2 แถว) → 1 แถว inline summary + toggle |

## 2.1 Workflow Progress Bar

> แสดงด้านบนสุดทุก role — เข้าใจ context ทันที

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  ●━━━━━━━●━━━━━━━●━━━━━━━○                                      │
│  บันทึก   ตรวจนับ  Auditor  Owner                               │
│   ✅       ✅       ⏳       ○                                   │

* ● = completed (สีเขียว), ⏳ = active (สีเหลือง/pulse), ○ = pending (สีเทา)

* ขนาด: h-8 total, compact single line

## 2.4 Collapsible Section Card (Pattern)

> ทุก section ใช้ pattern นี้เหมือนกัน

```
┌─────────────────────────────────────────────────────────────────┐
│  [icon] [title]    [status badge]    [1-line summary]   [▶/▼]  │  ← header
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  [expanded content]                                             │  ← body
│  (ซ่อนเมื่อ collapsed)                                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

Props:
  icon:       string (emoji)
  title:      string
  badge:      { label, variant } | null
  summary:    string (1 line, แสดงเมื่อ collapsed)
  expanded:   boolean (default: false)
  content:    slot

* คลิก header → toggle expand/collapse
* เมื่อ collapsed: แสดง summary text ข้าง badge
* เมื่อ expanded: ซ่อน summary, แสดง content
```

## Behavior Rules

```
1. Primary section: expanded by default — section ที่มี active form
2. Reference sections: collapsed by default — แสดง 1-line summary

4. Balance: compact 4 cards default, toggle แสดง 8 cards
5. Step1 active: ไม่ compact (layout เดิม — Manager ต้องเห็นทุกอย่าง)
