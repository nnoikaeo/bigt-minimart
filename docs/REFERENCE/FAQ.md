# ❓ Frequently Asked Questions (FAQ)

## General Questions

### Q: ระบบนี้ใช้ทำอะไร?
**A**: ระบบจัดการร้านมินิมาร์ทแบบครบวงจร สำหรับบันทึกรายรับ-รายจ่ายรายวัน ตรวจสอบ Audit และจัดทำรายงาน ทดแทนการใช้ Excel/Google Sheets

### Q: ใครสามารถใช้ระบบนี้ได้?
**A**: 5 บทบาท - เจ้าของร้าน, ผู้จัดการ, ผู้ช่วยผู้จัดการ, แคชเชียร์, Auditor (แต่ละคนมีสิทธิ์ต่างกัน)

### Q: ระบบมีค่าใช้งานไหม?
**A**: ไม่มี! ใช้ Firebase Free Tier ในระยะแรก ซึ่งเพียงพอสำหรับความต้องการของร้าน

### Q: ต้องมีอะไรเพื่อใช้ระบบ?
**A**: 
- คอมพิวเตอร์หรือ Tablet
- Internet connection
- Email address
- Browser (Chrome, Firefox, Safari, Edge)

## Technical Questions

### Q: ระบบใช้เทคโนโลยีอะไร?
**A**: Nuxt.js 3, Vue 3, TypeScript, Firebase, Tailwind CSS (เทคโนโลยีสมัยใหม่)

### Q: ข้อมูลจัดเก็บไว้ที่ไหน?
**A**: Firestore (NoSQL database) ของ Firebase ซึ่งเป็น cloud service ของ Google

### Q: ข้อมูลปลอดภัยไหม?
**A**: ✅ ปลอดภัยมาก - Firebase มีมาตรการความปลอดภัย Level Enterprise

### Q: สามารถ Backup ข้อมูลได้ไหม?
**A**: ✅ Firebase ทำ Backup อัตโนมัติทุกวัน ข้อมูลเก็บไว้ 30 วัน

### Q: สามารถ Offline ได้ไหม?
**A**: ❌ ไม่ได้ - ระบบต้องการการเชื่อมต่อ Internet

### Q: ลูกค้าอื่นสามารถเห็นข้อมูลของเราได้ไหม?
**A**: ❌ ไม่ได้ - ข้อมูลแต่ละร้านแยกกันแบบเสมอ

## Troubleshooting Questions

### Q: ไม่สามารถ Login ได้ - ทำไง?
**A**:
1. ตรวจสอบ email ถูกต้องไหม
2. ตรวจสอบ password ถูกต้องไหม
3. ลบ Cache และลอง login ใหม่
4. ติดต่อเจ้าของร้านถ้าปัญหาดำเนิน

### Q: หน้าเว็บไม่โหลด - ทำไง?
**A**:
1. Refresh หน้าเว็บ (Ctrl+R หรือ Cmd+R)
2. ลบ Cache และ Cookie
3. ลอง Browser อื่น
4. ตรวจสอบ Internet connection

### Q: บันทึกข้อมูลแล้วแต่ไม่เห็นในรายการ - ทำไง?
**A**:
1. รีเฟรชหน้าเว็บ
2. ตรวจสอบ Internet connection
3. ลอง Login ใหม่
4. ติดต่อ Developer ถ้า ปัญหายังอยู่

### Q: ได้ Error message - ทำไง?
**A**: 
1. อ่าน Error message ให้ดี
2. ตรวจสอบข้อมูลที่กรอก
3. ลอง reload หน้า
4. ถ่ายภาพ Error และติดต่อ Developer

### Q: ลืมกรอกช่องไหน - ทำไง?
**A**: ระบบจะแสดง Error ข้อมูลไม่ครบ เติมข้อมูลให้ครบแล้ว Save ใหม่

## Data Questions

### Q: สามารถแก้ไขข้อมูลได้ไหม?
**A**: ✅ ได้ - แต่ต้องตามสิทธิ์ (เช่น Manager ต้อง Submit ก่อน)

### Q: สามารถลบข้อมูลได้ไหม?
**A**: ✅ ได้ - แต่เฉพาะเจ้าของร้าน เท่านั้น

### Q: ข้อมูลหายไป - ทำไง?
**A**: ไม่ต้องกังวล Firebase บันทึก log ทุก action และมี Backup

### Q: สามารถลดเวลาบันทึกขอมูลได้ไหม?
**A**: ✅ ได้ - 
- ใช้แท็บ Key เพื่อย้ายช่องได้เร็ว
- ระบบจำข้อมูลเดิม (autofill)
- คลิก "Copy from yesterday" เพื่อคัดลอก
